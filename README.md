# Electronic Donate Blood Bank<br/>
 this is a project to help people getting blood so easy by searching for the boold type in specific area or everywhere around Yemen<br/>
  
  
 This project contains of :
 * Mobile Application 
 * Web Application 
 * API
 * Dashboard 
 * Database
